### tips
此项目需要先安装vue cli3
安装步骤：npm uninstall vue-cli -g      // 卸载vue-cli旧版本
npm install -g @vue/cli      // 安装vue-cli3.0
npm vue -V                  // 查版本
小编按照上面的步骤安装失败了，后来执行了以下的操作
执行npm cache clean --force
再安装以上步骤
